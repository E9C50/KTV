# KTV

#### 项目介绍

- KTV系统，C#前后台，Android客户端。播放器使用迅雷开源APlayer播放引擎。

#### 使用说明

- Aplayer：迅雷开源播放引擎，运行项目前导入此插件。
- DB：数据库，使用sqlserver 2008
- KTVPhoneManage：手机操控管理端
- MyKTVClient：前台娱乐端
- MyKTVServer：后台管理端
